GRooT Ag Grid

Wrapper for AgGrid

This library is a simple wrapper around the ag-grid library, that simplifies usage with Groot and offers various common utilities.

Getting started

Use the menu bar to access the various samples.

Installation

Assuming you already have installed Groot in your project, you need to install the library:

npm install @listgroup/groot-ag-grid ag-grid-angular ag-grid-community

In your angular module you will need to add a dependency on GrootAgGridModule.

In your main scss file, after having included Groot, add

@import "~@listgroup/groot-ag-grid/src/style/skin/darwin";
@import "~@listgroup/groot-ag-grid/src/style/groot-ag-grid";
